À propos

Ingénieur logiciel orienté gestion d'entreprise.

Je suis RAKOTONDRAZAKA Notahinjanahary Mirajo, développeur full-stack basé à Antananarivo, Madagascar. Mon terrain de jeu : les applications métier qui font tourner les entreprises — du web au desktop, en passant par le mobile et l'ERP.

Diplômé en Génie Logiciel & Base de Données à l'ENI Fianarantsoa, je construis des systèmes solides en PHP/Laravel, Python/Flask, Java/Spring Boot et C#/.NET, avec une vraie discipline d'architecture, de tests et de déploiement.

Ma signature, c'est Sage 100c — pas l'utilisateur final, le développeur. Je code directement contre les objets métier pour automatiser la facturation, la comptabilité et la gestion commerciale jusqu'au dernier maillon.

Autonome, rigoureux, orienté résolution de problèmes. Je travaille avec Git/GitHub, Docker et les pratiques DevOps modernes.

Précision & profondeur technique

Mes modules s'interfacent directement avec le moteur d'objets métier Sage pour automatiser des processus complets — de la saisie d'un document jusqu'à sa validation comptable et son écriture en base.

  • Objets métier — Document, Tiers, Article, Compte, Règlement (C#/.NET).
  • Automatisation — facturation, relances, écritures comptables générées en masse.
  • Requêtes & extractions — SQL natif et objets métier combinés.
  • Interfaçage — connecteurs tiers, import/export, migrations de données.

// Connexion à la base Sage 100c
var cptBase = new clsCptBase();
cptBase.Ouvrir(dossier, "ADMIN", motDePasse);

// Création d'un document de vente
var doc = new clsCptDocumentVente(cptBase);
doc.Tiers.CodeTiers = "CL00021";
doc.DateDocument    = DateTime.Now;
doc.AjouterLigne(article, qte: 12);

doc.Valider();  // → écriture comptable générée
cptBase.Fermer();
