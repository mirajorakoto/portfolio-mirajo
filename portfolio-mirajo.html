Architecture

Architecture logicielle

Mon mémoire M2 est une API REST C#/.NET professionnelle exposant Sage 100cloud via COM Business Objects — architecture multicouche conforme aux standards enterprise.

  • Clean Architecture (Domain, Application, Infrastructure, API)
  • STA Thread Pool — contrainte COM/STA Sage 100cloud
  • JWT Authentication + Refresh Token
  • Circuit-Breaker Pattern (résilience)
  • Rate Limiting & Concurrent Caching
  • Swagger / OpenAPI 3.0
  • Structured Logging (Serilog)
  • Reflection-based COM property access
  • TaskCompletionSource — async bridge
